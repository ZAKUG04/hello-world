# Gym Progress Tracker - Fitness Web Application

## Overview

This project is a multi-page fitness web application created as part of my software development course. The goal of this project was to apply HTML, CSS, and JavaScript to build a structured and interactive website that simulates a real gym platform.

The website includes multiple connected pages such as:
- Home (dashboard)
- Workout information page
- Gym machines guide
- Nutrition education page
- Subscription system (simulated payment system)

The main focus of this project was to improve my understanding of front-end development, page structure, and user interaction using JavaScript.

A key feature of this project is a simulated subscription system, where users can select a plan and complete a fake payment process to demonstrate how a real checkout system would work.

---

## Technologies Used

- HTML5 (structure and content)
- CSS3 (styling and layout)
- JavaScript (interactivity and page logic)
- Font Awesome (icons)
- Unsplash (free images for UI content)

---

## Key Features

- Multi-page website structure using HTML
- Responsive sidebar navigation across all pages
- Interactive subscription system with plan selection
- Simulated payment process (PayPal-style UI)
- Dynamic JavaScript functions connecting different parts of the site
- Educational content about workouts, machines, and nutrition

---

## What I Learned

Through this project, I learned:

- How to structure a multi-page web application
- How to use JavaScript to create interactive elements like modals and forms
- How to connect different pages using navigation systems
- How to simulate real-world systems like payment flows without backend logic
- How to improve UI/UX design for a more professional user experience

A particularly important learning experience was understanding how JavaScript can connect different pages and create a more dynamic user flow instead of static content.

I also learned how to simulate real-world features (like a subscription payment system), which helped me better understand how modern web applications work.
